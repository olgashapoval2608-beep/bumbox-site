# БУМБОКС — «Натисни PLAY, щоб прожити 20 років історії»

Інтерактивний односторінковий сайт-історія для гурту **Бумбокс**, побудований навколо однієї ідеї:
**магнітофон → касета → музика → час**. Користувач натискає **PLAY** і «проживає» 20 років історії
гурту через сім пов'язаних між собою сцен — як короткий інтерактивний музичний фільм.

> Концепція реалізована за технічним описом (DL Agency test brief): вінтаж + концерт +
> український рок/фанк/хіп-хоп настрій. Темно, кінематографічно, із зерном плівки та ламповим світлом.

## Сцени (один наскрізний скрол)

| # | Блок | Що відбувається |
|---|------|-----------------|
| 00 | **Hero** | Магнітофон проявляється з темряви під теплим прожектором. Кнопка **PLAY** оживляє його: крутиться касета, дисплей програє `TRACK 01 → ІСТОРІЯ → 2004 → сьогодні`, камера «влітає» в правий динамік → спалах-прожектор переходить в історію. |
| 01 | **Історія** | `2004 — Київ`. Текст про гурт, касетна стрічка-таймлайн, міні-аудіоплеєр (таймер біжить при скролі), фото гурту з пошаровим параллаксом. |
| 02 | **Альбоми** | `REWIND` → «Саундтрек нашого життя». Горизонтальна полиця, що рухається від вертикального скролу: центральний альбом збільшується, вініл висувається з конверта й обертається. Кнопка «Відкрити альбом» змінює настрій (колір) фону. |
| 03 | **Тури** | «Там, де нас чекали». Темна нічна карта зі світловими маршрутами (як FlightRadar), пульсуючі міста, картка концерту при наведенні, афіші найближчих концертів. |
| 04 | **Мерч** | «Забери музику додому». Backstage-стіл після концерту: вініл, футболка, кепка, квиток (перевертається), палички, пропуск — 3D-нахил при наведенні. Стрічка архівних кадрів. |
| 05 | **Моменти** | «Моменти між піснями». Архів розкиданих полароїдів, що оживають при наведенні. Велика цитата. Фінал: вогники натовпу збираються знизу, з'являється «Дякуємо, що були частиною цієї історії». |
| — | **Футер** | «Після концерту». Світний напис `БУМБОКС`, соцмережі-кнопки магнітофона, `TRACK 05 COMPLETED`, **Press PLAY again** (перемотка касети → повернення на початок), `2004 — ∞`. |

## Технології

Чистий **HTML + CSS + JavaScript**, без збірки та зовнішніх залежностей — відкривається будь-де.

- Увесь магнітофон, альбоми, мерч, карта й натовп намальовані на **CSS/SVG** (без зображень).
- Анімації керуються скролом через `IntersectionObserver` та один `requestAnimationFrame`-throttled обробник.
- Кастомний курсор-«PLAY» зі звуковою хвилею, плівкове зерно, повільні сценічні прожектори.
- Адаптив: на мобільному магнітофон займає весь перший екран, полиця альбомів стає вертикальною.
- Підтримка `prefers-reduced-motion` (анімації вимикаються) та доступність із клавіатури.

## Запуск

Просто відкрийте `index.html` у браузері. Для коректних шрифтів/локального сервера:

```bash
# будь-який статичний сервер, напр.:
python3 -m http.server 8000
# відкрийте http://localhost:8000
```

## Структура

```
index.html      — розмітка всіх 7 сцен
css/style.css   — палітра, кінематографічні оверлеї, усі анімації
js/script.js    — PLAY-перехід, карусель альбомів, карта, натовп, rewind
assets/         — місце для реальних медіа (поки порожнє)
```

## Як підставити реальний контент

Зараз фото/обкладинки/відео — стилізовані **заглушки**, щоб показати режисуру. Щоб додати реальні матеріали:

- **Фото гурту** (історія): замінити шари `.bandphoto__layer` на `<img>` (передній план / сцена / натовп).
- **Обкладинки альбомів**: у `.album__art` додати `<img>`; перевірити **дискографію та роки** (позначено `TODO` в `index.html`).
- **Карта турів**: координати міст задаються в `index.html` через `left/top` (%) — підставити реальні дати/майданчики.
- **Мерч / архівні кадри / моменти**: замінити CSS-заглушки та `.archive__ph` / `.photo` на справжні зображення; для «оживлення» фото при наведенні можна підставити коротке `<video muted loop>`.
- **Посилання**: соцмережі у футері та кнопки «Слухати / Квитки» — додати реальні URL.
